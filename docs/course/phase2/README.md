## 技术要点

### 数据库设计原则
1. **第三范式**：减少数据冗余，保证数据一致性
2. **软删除**：使用状态字段代替物理删除
3. **乐观锁**：版本号控制并发更新
4. **索引优化**：常用查询字段建立索引

### 核心技能
| 技能 | 应用场景 |
|:---|:---|
| MyBatis-Plus 多表关联 | 商品详情查询 |
| Spring Transaction | 订单创建、库存扣减 |
| Redis 缓存 | 商品信息、分类数据 |
| 文件上传 | 品牌Logo、商品图片 |
| 分页查询 | 商品列表、订单列表 |
